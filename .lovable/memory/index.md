# Project Memory

## Core
- Mobile-first, native iOS feel. Glassmorphism (#0F1419 bg), framer-motion springs, high-end animations.
- Typography: Inter (14px), sentence-case labels.
- Constraint: "Data first, decoration never." No emojis in headers, motivational copy, or gradient bgs. Zero "AI-template" aesthetics.
- Architecture: Supabase with strict Row Level Security (RLS) for data isolation.
- Localization: Built-in EN/NL support using external JSON translation files.
